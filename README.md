# Marvel App
O projeto consiste em consumir a API da Marvel e apresentar os quadrinhos, com algumas informações.

## Funcionalidades
    - Listar Quadrinhos
    - Ordenar Quadrinhos
    - Listar Detalhes do Quadrinho
    - Redirecionar para loja de quadrinhos no Google Maps
